# Gas Pass Backend - Installation & Utilisation

## Installation initiale

1. **Copier le fichier .env**
```bash
cp .env.example .env
```

2. **Installer les dépendances**
```bash
npm install
```

3. **Configurer la base de données PostgreSQL**
- Créer une base de données: `gaspass_db`
- Mettre à jour `.env` avec vos credentials

4. **Initialiser la base de données**
```bash
npm run db:sync
npm run db:seed
```

## Commandes

```bash
# Démarrage développement
npm run dev

# Démarrage production
npm start

# Sync tables BD
npm run db:sync

# Seed données initiales
npm run db:seed
```

## API Endpoints

### Products
- `GET /api/products` - Tous les produits
- `GET /api/products/:id` - Détails produit
- `POST /api/products` [Admin] - Créer produit
- `PUT /api/products/:id` [Admin] - Modifier produit

### Auth
- `POST /api/auth/register` - Inscription
- `POST /api/auth/login` - Connexion
- `POST /api/auth/logout` - Déconnexion

### Orders
- `POST /api/orders` [Auth] - Créer commande
- `GET /api/orders/:id` [Auth] - Détails commande
- `PUT /api/orders/:id/status` [Admin] - Changer statut

### Payment
- `POST /api/payment/create-intent` [Auth] - Créer Stripe Intent
- `POST /api/payment/webhook` - Webhook Stripe

### Telegram
- `POST /api/telegram/webhook` - Webhook Telegram Bot
- `POST /api/telegram/link` - Lier compte Telegram

### Admin
- `GET /api/admin/dashboard` [Admin] - Stats
- `GET /api/admin/orders` [Admin] - Toutes commandes
- `GET /api/admin/products` [Admin] - Tous produits
- `POST /api/admin/products` [Admin] - Créer produit
- `GET /api/admin/users` [Admin] - Tous utilisateurs

## Variables d'environnement requises

```
DB_HOST=localhost
DB_PORT=5432
DB_NAME=gaspass_db
DB_USER=postgres
DB_PASSWORD=...

JWT_SECRET=...
STRIPE_SECRET_KEY=...
TELEGRAM_BOT_TOKEN=...
```

## Notes

- Token JWT: 15 minutes
- Refresh token: 7 jours
- Middleware auth.js protège les routes
- Telegram webhook mode (pas polling)
